В данной программе не реализованы этапы проверки (вводимой информации от пользователя) на корректность (кроме проверок на ввод доступных операций и критериев поиска/сортировки).
Все типы вводимых полей для объекта Товар являются строковыми, кроме цены. Все строковые поля принимают информацию через getline.
Дату необходимо вводить строго в указанном формате DD.MM.YYYY
Реализация проверок на корректность для вводимых полей рассматривается как дальнейшее продолжение данного проекта.
